SNTNEL — Sentinel Drone Network

AI-Powered Autonomous Emergency Response Ecosystem


Overview

SNTNEL (Sentinel Drone Network) is an AI-powered emergency response ecosystem designed to reduce emergency response time using autonomous drones, real-time tracking, intelligent safety monitoring, and live emergency coordination.

The system integrates a smart safety platform with autonomous aerial support capable of assisting civilians during emergencies such as accidents, harassment, medical emergencies, and disaster situations.


Problem Statement

In critical emergencies, delayed response often results in severe consequences. Traditional emergency systems face challenges such as:

* Delayed response time
* Lack of live situational awareness
* Difficulty sharing accurate location
* Limited real-time evidence collection
* Slow coordination between civilians and authorities

SNTNEL addresses these issues using autonomous drone deployment and intelligent emergency coordination.

The hardware system includes:

* Pixhawk Flight Controller
* GPS Navigation Module
* Brushless Motors
* Electronic Speed Controllers (ESC)
* Li-Po Battery System
* Power Distribution Board
